# carService

Личное приложение для учета ТО, ремонтов и расходов по автомобилю.

## Быстрый старт

```bash
npm install
npm run dev
```

## Документация

- Требования и модель данных: `documents/ReadMe.md`
- План работ по модулям: `documents/RoadMap.md`

## PWA (установка на iPhone)

Приложение можно добавить на экран «Домой» как PWA. Данные хранятся локально в браузере (IndexedDB), отдельный сервер для журнала не нужен.

### Сборка и публикация

```bash
npm run build
```

Загрузите содержимое папки `dist/` на статический хостинг с HTTPS (например, Cloudflare Pages, Vercel, Netlify).

### Установка на iPhone

1. Откройте опубликованный адрес в **Safari**.
2. Нажмите **Поделиться** → **На экран «Домой»**.
3. Запускайте приложение с иконки на рабочем столе.

### Локальная проверка PWA

```bash
npm run build
npm run preview -- --host
```

С телефона в той же Wi‑Fi сети можно открыть `http://<IP-вашего-ПК>:4173` для проверки вёрстки. Для полноценной установки PWA на iOS нужен HTTPS (деплой на хостинг).
